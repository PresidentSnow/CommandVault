# Automated tasks
Folder for store the scripts for automate tasks in Linux.

## Distributions and their structure:

* Arch-based distributions:
    - Docker and docker compose, it's installation and uninstallation.
    - VS Code, it's installation and uninstallation.

* Debian-based distributions:
    - Docker and docker compose, it's installation and uninstallation.
    - VS Code, it's installation and uninstallation.

## Ops toolkits:

* Backups

* Network scripts:
    - Scan localnet (LAN)

* userManagement:
